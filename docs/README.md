# Documentation Index

## Folder Layout

- `docs/resume/`
  - Resume feature docs, architecture, maintenance, and PDF layout baseline files.
- `docs/specs/`
  - Long-form technical specs for major features.
- `docs/ops/`
  - Operational runbooks and recovery procedures.
- `docs/prompts/`
  - Reusable AI prompts and prompt workflows.
- `docs/archive/`
  - Historical/temporary artifacts kept for reference.

## Common Entry Points

- Resume quick start: `docs/resume/README_RESUME.md`
- Resume technical architecture: `docs/resume/RESUME_ARCHITECTURE.md`
- Resume maintenance: `docs/resume/RESUME_MAINTENANCE.md`
- Resume PDF layout constraints: `docs/resume/resume-generation-spec.md`
- Deploy recovery runbook: `docs/ops/DEPLOY_RECOVERY.md`
- Contact form spec: `docs/specs/CONTACT_FORM_SPEC.md`
- CDN deployment spec: `docs/specs/CDN_DEPLOYMENT_SPEC.md`
